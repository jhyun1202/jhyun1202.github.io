## Overview

Developing acoustic-based sensor system to detect and analyze fuel sloshing behavior inside opaque metallic rocket tanks.

This research addresses critical safety concerns in aerospace applications where visual monitoring is impossible during flight operations.

## Key Contributions

- Built custom acoustic sensor arrays
- Developed signal processing algorithms to identify dangerous sloshing patterns
- Testing various tank geometries and fill levels
- Creating comprehensive detection models

## Applications

Research has applications beyond rocketry, including:
- Commercial aviation fuel systems
- Industrial fluid transport
- Rocket flight stabilization

**Status:** Ongoing Research
